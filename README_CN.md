# Whisper Hotkey（Windows 语音输入工具）

一个基于 **faster-whisper** 的 Windows 全局快捷键语音转文字工具。

按下快捷键，说话，松开 —— 文字会自动识别并粘贴到当前窗口。

---

## 功能特点

- 全局快捷键（无需切换窗口）
- 基于 Whisper 的高准确率语音识别
- 模型下载完成后可完全离线使用
- 简洁界面，内置赞助二维码
- 免安装、绿色便携

---

## 系统要求

- Windows 10 / Windows 11（64 位）
- 可用麦克风（内置或外接）
- **首次运行需要联网**

---

## 下载与运行

1. 前往 GitHub 的 **Releases** 页面下载最新版
2. 解压压缩包
3. 双击运行 `whisper_hotkey.exe`

> ⚠️ **重要提示：**  
> 请不要删除、移动或重命名文件夹中的任何文件。  
> 程序必须在当前目录结构下运行。

---

## 关于首次启动（非常重要）

### 为什么第一次启动 / 第一次识别会很慢？

这是**正常现象**，不是程序卡死。

在第一次启动以及第一次进行语音识别时：

- 程序会自动下载 Whisper 语音模型
- 模型体积较大（数百 MB）
- 根据网络和电脑性能，可能需要 **1～3 分钟**

模型下载完成后：

- 以后启动会明显加快
- 语音识别速度会非常快

> ⚠️ 模型只需下载一次。

---

## 使用方法

1. **按住 F8** → 听到提示音，开始录音
2. 对着麦克风说话
3. **松开 F8** → 听到提示音，结束录音
4. 识别结果会自动粘贴到当前输入框

---

## 麦克风与权限说明

- 第一次运行时，Windows 可能会询问麦克风权限  
  → 请点击 **允许**
- 如果没有声音或识别失败，请检查：
  - Windows 麦克风隐私设置
  - 是否选中了正确的输入设备

---

## 离线使用说明

- 仅**第一次运行**需要联网下载模型
- 模型下载完成后，可完全离线使用

---

## 常见说明

- 第一次识别速度会比后续慢
- 杀毒软件可能会提示“未知程序”  
  （这是未签名 EXE 的常见情况）
